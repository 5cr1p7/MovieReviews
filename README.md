# MovieReviews
Android Application with movie reviews using API from NY Times (https://developer.nytimes.com/docs/movie-reviews-api/1/overview)

## Technologies:

- MVVM
- Kotlin
- Kotlin Coroutines
- ViewBinding
- Single Activity
- Retrofit
- Moshi
